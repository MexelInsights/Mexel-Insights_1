# Mexel Insights

Geopolitical intelligence platform providing critical materials supply chain analysis.

## Tech Stack

- **Frontend:** Multi-page site in `public/` — 7 HTML pages, shared CSS (`css/styles.css`) and JS (`js/shared.js`, `js/tools.js`, `js/map.js`, `js/feed.js`)
- **Pages:** `/` (Home), `/scenario-lab`, `/materials-watch`, `/policy-monitor`, `/risk-map`, `/ai-tools`, `/pricing`
- **Backend:** `server/index.js` — Express + Anthropic API proxy
- **Hosting:** Render
- **Domain:** mexelinsights.com
- **Deployment:** Git push to `main` auto-deploys on Render

## Design Direction

- Style: bright, modern, premium
- Backgrounds: light
- Text: navy
- Accents: copper
- Fonts: Instrument Serif, DM Sans, JetBrains Mono

## Data Coverage

- 30 critical materials
- 14 jurisdictions
- 12 sectors
